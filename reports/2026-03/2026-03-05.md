# 日报 2026-03-05

## 论文笔记
- 论文：AReaL: A Large-Scale Asynchronous Reinforcement Learning System for Language Reasoning
- 链接：https://arxiv.org/abs/2505.24298v5

### 关键点
- 问题：同步式 RL（rollout→训练）被最长输出拖慢，推理长度长尾导致 GPU 空转。
- 系统：生成与训练彻底解耦；rollout workers 流式生成，trainer 凑够 batch 就更新；权重更新可打断生成（interruptible）。
- 算法：用 staleness 控制(η)限制样本陈旧；用 decoupled PPO/近端策略 π_prox 处理异步与多版本轨迹，稳定训练。
- 结果：在数学/代码推理上最高 ~2.77× 训练加速，扩展到 512 GPU，最终性能不降甚至略升。
- 局限：工程复杂度高；中断带来 KV 重算开销；η 需要调参；reward 服务/单测稳定性是命门。

### Take
- 我的 take：这篇核心价值是“系统-算法共设计”把异步 RL 变成可控旋钮（吞吐 vs staleness），对推理型后训练非常实用；思路可迁移到我们做的自动化流程：拆解瓶颈、异步流水线、幂等与可观测。
